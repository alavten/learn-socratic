# 第二章 数学与工程基础

## 一、图论应用

### 1. 最小生成树

- **定义**：连通赋权图的所有生成树中，边的权值之和最小的生成树称为最小生成树（包含图中所有顶点）。
- **算法**：**Prim 算法**（最近顶点）、**Kruskal 算法**（最短边）。
- **解题要点**：每次找最短边，直到所有结点连成一棵树且无回路。

### 2. 最短路径

- **定义**：赋权图中的最短路径问题，是求两顶点之间路径长度最小的路径。长度指路径上各边权值之和，而不是边的条数；长度的具体含义取决于边权所代表的意义。
- **算法**：**Dijkstra 算法**。
- **解题要点**：从源点开始，计算到下一结点的最短路径；在后续分析中，对当前结点只考虑前一结点的最短路径长度，直至到达终点位置，再分析最短路径。

### 3. 网络与最大流量

- **背景**：许多应用涉及流量问题，如道路系统中的车流、控制系统中的信息流、网络系统中的数据流、金融系统中的现金流等。
- **定义**：在实际应用中，常需求解最大流问题；它是一个**特殊的线性规划问题**。
- **解题要点**：每次从起点到终点找出并抽出一条最大流路径，直到不存在可达路径为止；所抽出的流量之和即为最大流。

```text
起点 ==========> …… ==========> 终点
      反复：找最大流路径并抽出，直到无可达路径
      累计抽出流量 = 最大流
```

## 二、预测和决策

### 1. 线性规划

#### （1）线性规划的一些特点

1. 线性规划的可行域由一组线性约束形成；从几何上看，是由若干线性平面所围成的区域，**不一定是封闭的多边形或多面体**。
2. 若存在两个最优解，则连接这两点的**线段上的所有点**都是最优解；而线段的**延长线**可能超出可行域。
3. 增加一个约束时，要么**缩小**可行域（新约束划分原可行域），要么可行域**不变**（新约束与原可行域不相交）。
4. 若在可行域边界的**非顶点**处达到最优解，则当目标函数沿某一方向移动时，目标函数值将增大或减小（与最优解矛盾），或**保持不变**（在该边界线段上所有点均达到最优）。

#### （2）线性规划的标准型

- 目标函数为**极大化类型**；
- 所有的约束条件都是**等式**；
- 所有约束方程右端的常数都是**非负**的；
- 所有决策变量都是**非负**的。

### 2. 动态规划

- **定义**：动态规划是求解决策过程最优化的过程。
- **解题关键**：可通过暴力法/穷举法，将所有方案列举出来，求最优解。也可以通过贪心策略，找到最初方案再进行调整，直到找到最优解。

### 3. 风险型决策

- **决策树分析**：决策树分析是一种用于分析决策过程的方法。它通过将决策过程建模为一棵决策树，从而明确各个决策节点的决策选项和相应的风险和效益，进而帮助决策者做出最佳决策。

## 三、数学建模

### 1. 概念

- 数学建模是一种数学的思考方法，是运用数学的语言和方法，通过抽象和简化，建立能近似刻画并解决实际问题的模型的一种强有力的数学手段。

### 2. 数学建模过程

- **模型准备**：了解问题的实际背景，用数学语言来描述问题。
- **模型假设**：根据实际对象的特征和建模的目的，对问题进行必要的简化，并用精确的语言提出一些恰当的假设。
- **模型建立**：在假设的基础上，建立相应的数学结构。
- **模型求解**：利用获取的数据资料，对模型的所有参数做出计算（估计）。
- **模型分析**：对所得的结果进行数学上的分析。
- **模型检验**：将模型分析结果与实际情形进行比较，以此来验证模型的准确性、合理性和适用性。
- **模型应用**：应用方式因问题的性质和建模的目的而异。

### 3. 数学建模方法

- **（1）直接分析法**：理解原理，直接构造模型。
- **（2）类比法**：根据类似问题的模型构造新模型。
- **（3）数据分析法**：在对大量数据进行统计分析之后建立模型。
- **（4）构想法**：通过对未来可能发生的情况做出假设来建立模型。

### 4. 模型分析

- **模型的合理性分析**：包括**最佳**、**适中**、**满意**等。
- **模型的误差分析**：包括模型误差、观测误差、截断误差、舍入误差、过失误差、绝对误差、相对误差等。
- **参数的灵敏性分析**：分析变量数据是否敏感，以及在最优解不变的条件下，这些变量容许变化的范围。

### 5. 模型检验

- 利用实际案例数据来检验模型。
- 邀请专家分析模型是否合理。
- 用计算机模拟实际问题，然后在计算机上验证数学模型。

## 四、随机函数

**概率密度函数**：设随机变量 s 的分布密度函数为 f(x) = 2x。当 Δx 充分小时，随机变量 s 落在区间 (x, x + Δx) 内的概率近似等于 f(x)Δx。

**伪随机数**：设 r₁、r₂ 为计算机生成的、相互独立且在区间 (0, 1) 上均匀分布的伪随机数。

**要点**：

- 随机数近似等于密度函数的结果。
- 若密度函数呈上升趋势，随机数也应满足上升趋势。
- 使用仿真函数 max(r₁, r₂) 随机生成随机数 r₁、r₂。
